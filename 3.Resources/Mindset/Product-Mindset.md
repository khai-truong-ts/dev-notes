Bạn có thể dùng bảng này để training team:

|**Hoạt động**|**Tư duy truyền thống (Code Mindset)**|**Tư duy sản phẩm (Product Mindset)**|
|---|---|---|
|**Nhận task**|"Tôi cần làm gì?"|"Tại sao chúng ta làm việc này? Vấn đề của user là gì?"|
|**Design**|"Tôi sẽ code y hệt design."|"Design này có tối ưu cho mobile không? Có làm chậm trang không?"|
|**Code**|"Tôi tập trung vào logic, làm cho nó chạy."|"Tôi tập trung vào performance, A11y, và các trạng thái (loading/error/empty)."|
|**Testing**|"Tôi test 'happy path' trên máy tôi."|"Tôi test như một người dùng thật, cố gắng 'phá' nó, test trên 3G, trên thiết bị cũ."|
|**Code Review**|"Review logic code."|"Review cả logic code VÀ trải nghiệm người dùng (UX/UI)."|
|**Hoàn thành**|"Code đã được merge."|"Tính năng đã release, chạy ổn định, và user _thực sự_ dùng nó để giải quyết vấn đề."|
